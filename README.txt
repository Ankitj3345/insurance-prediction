🏥 Health Insurance Cost Prediction
📌 Project Overview
The Health Insurance Cost Prediction System is a Machine Learning web application that predicts medical insurance charges based on personal and health-related information.

Users can enter details such as age, gender, BMI, number of children, smoking status, and region to estimate their insurance costs.

The application is built using Python, Scikit-Learn, and Streamlit.

🎯 Objective
The objective of this project is to analyze the factors affecting medical insurance costs and build a predictive model that can estimate insurance charges accurately.

📊 Dataset Features
The model uses the following features:

Age
Sex
BMI (Body Mass Index)
Number of Children
Smoking Status
Region
Target Variable:

Charges (Medical Insurance Cost)
